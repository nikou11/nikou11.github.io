# Noki – Language Learning App  
### UX Design Case

**Role:** UX Design  
**Focus:** Language learning, interaction design, user research

Noki is an interactive language learning experience designed around **real-world conversations rather than memorization drills**.

### Problem

Many language learning apps emphasize vocabulary and grammar drills but fail to help users practice real-life communication.

### Research

Conducted interviews with language learners to identify challenges with traditional learning tools.

Key insights:

• learners struggle applying vocabulary in real conversations  
• users want quick daily practice sessions  
• conversation practice is the biggest unmet need

### Solution

A scenario-based language learning platform where users practice everyday situations through interactive story-led lessons.

Examples:

- ordering food at a restaurant  
- asking for directions  
- social conversations  
- daily challenges with community interaction

### Design Process

1. User interviews  
2. Personas  
3. Information architecture  
4. User flows  
5. Wireframes  
6. Prototype design

# Honours Thesis  
### Lightness Constancy in Virtual Reality

My honours thesis investigates **how observers estimate surface reflectance under changing illumination conditions**.

Using a controlled **psychophysical experiment in virtual reality**, participants adjust a matching patch while the orientation of a reference surface changes relative to a light source.

### Research Focus

- visual perception  
- illumination inference  
- reflectance perception  
- lightness constancy

### Methodology

- experimental design using repeated trials  
- virtual environments with controlled lighting conditions  
- perceptual matching tasks  
- statistical analysis of observer responses

This research contributes to understanding how humans interpret visual scenes when lighting conditions change.
